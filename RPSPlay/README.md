<h4 align="center">RPSPlay</h4>
<p align="center">RPSPlay is a modern and engaging rock, paper, scissors game designed for quick, fun, and interactive gameplay. Built with HTML, CSS, and JavaScript, it offers a clean interface, responsive design, and smooth user experience that makes every match enjoyable. Whether you’re playing for fun or testing your strategy, RPSPlay delivers a simple yet entertaining experience.</p>
